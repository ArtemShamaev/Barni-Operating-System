# Barni Operating System
Добро пожаловать в отечественную ОС BarniOS! Здесь можно посмотреть исходный код проекта и много другой полезной информации! 
________________________________________________________________________________________________________________________________________________________________
                           УСТАНОВКА ЯДРА И ЗАГРУЗЧИКА
________________________________________________________________________________________________________________________________________________________________

Для установки системы Barnino Systems BarniOS необходимо скачать с этого сайта (https://github.com/ArtemShamaev/Barni-Operating-System) все файлы (исходный код) 
нажав на кнопку Code -> Download ZIP. После скачивания, нужно распаковать папку которую Вы скачали.



________________________________________________________________________________________________________________________________________________________________________
                                УСТАНОВКА И КАСТОМИЗАЦИЯ ЧЕРЕЗ MAKEFILE
________________________________________________________________________________________________________________________________________________________________________
Можно использовать утилиту make. Установка отличается от Вашей системы

                                ==============================================================
                                ========    Для Windows 10, Windows 11 Pro/Home    ===========
                                ==============================================================

В меню "Пуск" (его поиске) введите Powershell. Кликните по иконке Powershell(а) (Символы _> на синем фоне) ПРАВОЙ кнопкой мыши. Нажмите "Запуск от имени адменистратора". После, возможно, у вас спросят пароль адменистратора и потвеждение операции (он спросит "Разрешить вносить измениения на устройство приложению Powershell"), нажмите "Да". После, откроется окно командной строки Powershell. Введите в нем 
# Set-ExecutionPolicy Bypass -Scope Process -Force; [System.Net.ServicePointManager]::SecurityProtocol = [System.Net.ServicePointManager]::SecurityProtocol -bor 3072; iex ((New-Object System.Net.WebClient).DownloadString('https://community.chocolatey.org/install.ps1'))

После, введите choco install make. Теперь переходите к пункту "Начало работы с make".

                                =============================================================
                                ======== Для Debian, Ubuntu, Linux Mint и подобных ==========
                                =============================================================

Откройте меню "Пуск", введите в поиске "Терменал" (или "Консоль"), откройте это приложение (иконка ввиде черного окна и нескольких символов на нем). В нем введите команду
# sudo apt install make
Теперь переходите к пункту "Начало работы с make".

                               ===============================================================
                               ============== Начало работы с make ===========================
                               ===============================================================

Чтобы усатновить систему BarniOS и запустить её нужно сначала скачать BarniOS с этого сайта (см. выше). Потом зайдите в ваш терменал, консоль или Powershell (см. выше).
# Для Windows
Введите cd C:\Users\Имя_Пользователя\Downloads. Вместо Имя_Пользователя подставте Ваше имя пользователя. Например cd C:\Users\Maxim\Downloads
# Для Linux
Введите cd /home/Имя_Пользователя/Загрузки. Вместо Имя_Пользователя подставте Ваше имя пользователя. Папка Загрузки может называться как Downloads в зависемости от языка системы. Проверьте оба варианта.

После, введите (вне зависемости от целевой системы)
# ВНИМАНИЕ: make download использовать только в Линукс. В Windows нужно отдельно скачать nasm и QEMU.
# make download
# make compile
# make run
make download установит зависемости (нужно запускать от имени адменистратора в Windows), make compilate компилирует ядро и загрузчик, make run запустит систему в QEMU.
Для кастомизации системы нужно ввести 
# make config
Откроется приглашение ввода Barni Config Changer. В нем введите нужные команды. Вот команды, которые Вы можите ввести:
# exit - выйти и сохранить
# exitws - выйти без сохранения
# usr_name новое_имя - сменить имя полькователя (вместо новое_имя введите новое имя пользователя)
# change_password новый_пароль - сменить пароль (вместо новый_пароль введите новый пароль).
