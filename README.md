# Barni-Operating-System
Добро пожаловать в отечественную ОС BarniOS! Здесь можно посмотреть исходный код проекта и много другой полезной информации! 
________________________________________________________________________________________________________________________________________________________________
                           УСТАНОВКА ЯДРА И ЗАГРУЗЧИКА
________________________________________________________________________________________________________________________________________________________________

Для установки системы Barnino Systems BarniOS необходимо скачать с этого сайта (https://github.com/ArtemShamaev/Barni-Operating-System) все файлы (исходный код) 
нажав на кнопку Code -> Download ZIP. После скачивания, нужно распаковать папку которую Вы скачали.

_________________________________________________________________________________________________________________________________________________________________
                                      ЗАПУСК В QEMU
_________________________________________________________________________________________________________________________________________________________________

Для тестирования системы, которую Вы скачали нужно сделать действия, которые ОТЛИЧАЮТСЯ в зависемости от Вашей целевой системы.

                                ==============================================================
                                ========    Для Windows 10, Windows 11 Pro/Home    ===========
                                ==============================================================
                                
Открыть «Проводник» Windows и перейти к директории, где находится win_qemu.bat.
Кликнуть по этому файлу правой кнопкой мыши (win_qemu.bat), нажать "Запуск от имени адменистратора". Затем потведить (система может запросить от Вас пароль адменистратора). После этого установится компилятор языка ассемблера и QEMU, откроется окно эмулятора и Вы сможите наслаждаться тестированием отечественной системы!
Если Ваш курсор мыши изчез, нажмите Ctrl+Alt+G. Спасибо за выбор BarniOS!

                                =============================================================
                                ======== Для дистрибутивов Linux Ubuntu, Linux Mint =========
                                =============================================================
                                
Откройте меню "Пуск", введите в поиске "Терменал", откройте это приложение (иконка ввиде черного окна и нескольких символов на нем). В терменале введите несколько команд: cd /путь/к/установленной/системе, потом введите chmod +x ./UbuntuQemu.sh и ./UbuntuQemu.sh


Для установки на другие ОС введите в комментариях под видео на https://www.youtube.com/watch?v=bBVZFNU_8Ik&t=3s. Буду очень рад если Вы подпишетесь на мой youtube-канал!
